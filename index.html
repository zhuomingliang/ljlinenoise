Overview

ljlinenoise is a pure LuaJIT port of linenoise, a small alternative to readline and libedit.

ljlinenoise is based on ljsyscall.

ljlinenoise is compatible with lua-linenoise.

Status

ljlinenoise is in beta stage.

It's developed for LuaJIT 2.0.

Download

ljlinenoise source can be downloaded from GitHub.

Installation

The easiest way to install ljlinenoise is to use LuaRocks:

luarocks install ljlinenoise

or manually, with:

make install
